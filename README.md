#  Personal Portfolio Website

A responsive personal portfolio website built using **HTML** and **CSS** to showcase my profile, skills, certifications, and achievements. This project reflects my learning journey in web development and my interest in creating clean, user-friendly web interfaces.
##  About the Project

This portfolio website is designed to present my academic background and technical skills in a simple and professional way. It includes multiple sections such as Home, About Me, Skills, Certifications, Achievements, and Contact.

The project focuses on:
- Clean UI design
- Responsive layout
- Structured HTML and CSS
- Beginner-friendly implementation
##  Technologies Used

HTML5 – Structure of the website  
CSS3 – Styling and layout  
Remix Icon – Icons for social links  
Google Fonts (Poppins) – Typography  

---

## Features

- Responsive personal portfolio layout  
- About Me section with profile image  
- Skills displayed using tags  
- Certifications and achievements listing  
- Contact form UI  
- Social media icons (GitHub, LinkedIn, Instagram)
